## Technical Overview

This website utilizes the following technologies for creation and deployment:

### **Website Creation**
* **Framework:** **Quarto** (`.qmd` files)
* **Analysis:** **R** with the **Tidyverse** ecosystem (`dplyr`, `tidyr`, `ggplot2`)
* **Goal:** To analyze the longitudinal trends in **popularity** and **user satisfaction** across various movie genres.

### **Deployment**
* **Platform:** **GitHub Pages**
* **Output Folder:** All HTML files and assets are rendered into the **`/docs`** directory, which serves as the source for the deployment.
* **Version Control:** Git
